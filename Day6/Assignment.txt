Difference between git init and git clone.
Git init initializes a new Git repository in an empty or existing directory where as a git clone creates a copy of an existing remote repository, including its history and files.

Difference between git branch and git checkout.
git branch is used to list, create, or delete branches, while git checkout is used to switch between branches.

Difference between git fetch and git pull.
git fetch downloads updates from a remote repository but does not merge them into your local branch, while git pull fetches and automatically merges the changes into your current branch.

Show the difference between git reset and git revert.
git reset moves the branch pointer and potentially removes commits, altering history, while git revert creates a new commit that undoes changes without changing history.
